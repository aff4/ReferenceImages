Evimetry Pre Standard Images

These images are produced using the Evimetry 2.0/2.1 AFF4 implementation. 

	Base-Linear.af4				:	Linear image of the entire source device. 
	Base-Linear-ReadError.af4	: 	Linear image of an entire source device, where a read error occurred. 
	Base-Allocated.af4			: 	Linear image of volume & FS metadata, and all allocated blocks in the filesytem.
